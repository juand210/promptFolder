Preguntas Frecuentes

¿Cómo pasar todo tu código a ChatGPT o Claude de una sola vez?

Arrastra tu carpeta raíz para generar un archivo consolidado. Este contendrá el árbol visual del repositorio y el código de cada archivo envuelto en delimitadores claros (--- INICIO ARCHIVO --- y --- FIN ARCHIVO ---) que la IA interpretará perfectamente.

¿Cómo dar el contexto completo de un repositorio a Gemini?

Usa esta herramienta para generar el archivo de texto estructurado de tu proyecto y luego súbelo directamente al chat o pégalo en el prompt de Gemini. Su amplia ventana de contexto leerá toda la estructura y el código fuente de inmediato.

¿Es seguro cargar mis archivos y carpetas de código aquí?

Sí, 100% seguro. El procesamiento se ejecuta por completo en tu navegador (Client-side JavaScript). Tus archivos jamás se envían a servidores ni se almacenan en ninguna base de datos; todo ocurre localmente en tu ordenador.

¿Cómo estima los tokens la herramienta?

Utiliza una aproximación estándar y veloz de 1 token equivalente a 4 caracteres de texto plano o código en inglés. Esto te permite tener una referencia inmediata del consumo de contexto sin ralentizar el navegador.

¿Qué archivos y carpetas se excluyen automáticamente?

Se omiten por defecto las carpetas de dependencias y temporales (node_modules, .git, venv, dist, .next), archivos de bloqueo (package-lock.json, etc.) y ficheros multimedia o binarios comunes para optimizar el límite de contexto.
